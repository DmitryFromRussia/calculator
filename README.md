calculator - проект-приложение ввиде простого калькулятора, производящего примитивные вычисления, таки, как:
'+' - сложение чисел
'-' - вычитание чисел
'*' - перемножение чисел
'**' - возведение числа в стпень
'/' - деление чисел 
'//' - целочисленное деление чисел
'()' - скобки
'C' - функция очистки поля
